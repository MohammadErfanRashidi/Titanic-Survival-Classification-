# Titanic Classification

This project is a machine learning classification task based on the famous Titanic dataset.  
The goal is to predict whether a passenger survived the Titanic disaster using features such as age, sex, class, and more.

## Project Overview
- Loads and explores the Titanic dataset
- Performs data cleaning and preprocessing
- Applies feature engineering (e.g., handling missing values, encoding categorical variables)
- Trains and evaluates classification models
  
## Requirements
- Python 3.x
- Jupyter Notebook
- Common data science libraries:
  - pandas
  - numpy
  - matplotlib / seaborn
  - scikit-learn
